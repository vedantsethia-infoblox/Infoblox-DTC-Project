About

  • This solution aims at dynamically changing the DTC pool ratios at the LBDN level based on parameters like CPU Utilization that can be monitored using SNMP.

  • It intelligently modifies the ratios to the optimum configuration once the specified threshold is reached.

Note: This solution does not apply to LBDNs with a single pool or LBDNs with pools that have the same ratio.
